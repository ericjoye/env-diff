# env-diff — Pricing

## Model: Open Source (Free)

env-diff is **free and open source** (MIT licensed). This is intentional — it positions the tool as a widely-adopted developer utility and builds goodwill in the Python/DevOps community.

## Rationale

- Developer tools with zero friction (free, zero deps) get adopted fastest
- A free CLI tool serves as a portfolio piece and funnel for future paid tooling
- GitHub Sponsors / funding can provide optional revenue without gating features

## Future monetization (post-MVP, optional)

- **env-diff Pro** (if demand warrants): `--json` output, `--ignore` flag, multi-line value support, diff across 3+ files, watch mode
- **GitHub Sponsors** for those who want to support maintenance
- **Consulting** for teams needing custom env management tooling

## Current ask

$0. Just install it and use it.
