# Privacy Policy — env-diff

**Last updated:** June 20, 2026

## Overview

env-diff ("we", "our", "the tool") is a command-line utility that compares two `.env` files and shows differences in their key-value pairs, ignoring order and comments.

## Data Collection

**env-diff does NOT collect, store, or transmit any data.**

When you use env-diff:

- **No data is logged or stored.** All comparison happens in memory. The tool does not write temporary files, logs, or databases.
- **No network calls.** env-diff does not make any network requests. It operates entirely offline.
- **No telemetry.** There is no usage tracking, analytics, or telemetry of any kind.
- **No accounts.** There is no signup, login, or user account system.

## How It Works

env-diff reads the two `.env` files you specify as arguments, parses their key-value pairs (skipping comments and blank lines), and outputs a color-coded diff to stdout. It does not modify either file. Your data is never written to disk by the tool.

## Open Source

env-diff is open source under the MIT license. It has zero runtime dependencies — it uses only Python 3.11+ standard library modules.
